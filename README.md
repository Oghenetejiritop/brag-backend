# BRAG Backend

BRAG (Bot Retrieval-Augmented Generator) is a backend service that enables users to upload and process documents, then query them through a Retrieval-Augmented Generation (RAG) pipeline.

The system supports multiple data sources and exposes API endpoints for building custom, domain-specific chatbots powered by user-provided knowledge.

---

## Features (Phase 1)

- Multi-format document ingestion (PDF, TXT, DOCX, and web content)
- Text preprocessing and chunking
- Embedding generation using LLM providers
- Semantic search with vector databases
- Context-aware responses using LLMs
- API-ready architecture for chatbot integration

## Supported Data Sources (Planned / Incremental)

- PDF files
- Plain text files (.txt)
- Word documents (.docx)
- Web pages (via URL ingestion)

---

## Tech Stack

- FastAPI
- LangChain (Core, Community, OpenAI)
- ChromaDB (Vector Store)
- OpenAI Embeddings & LLMs

---

## Project Goal

To build a scalable, backend-first platform that allows users and businesses to create custom AI-powered chatbots by uploading their own data and accessing it through API endpoints.
